### Description
Unity-Headunit is an Unity3D Game Engine based headunit using C# as its main component + C++ (arduino). Main goal is to create a universal headunit that runs on any x86 linux system (not rpi).

### Supported Systems
 - [YouYeeToo X1 x86 SBC](https://amazon.com/dp/B0CCY2RBCS/)
 - Any x86 / x64 System

### Supported functionalities
 - 480p, 720p and 1080p with 30 or 60 FPS
 - RaspberryPI 3 hardware acceleration support to decode video stream (up to 1080p@60!)
 - Audio playback from all audio channels (Media, System and Speech)
 - Audio input for voice commands
 - Touchscreen and buttons input
 - Bluetooth
 - Automatic launch after device hotplug
 - Automatic detection of connected Android devices
 - Wireless (WiFi) mode via head unit server (must be enabled in hidden developer settings)
 - User-friendly settings

### Supported platforms

 - Ubuntu
 - Windows
